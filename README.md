## Comandos de Decisão

Comandos de decisão são fluxos de escolha em que o programa deve escolher entre dois ou mais caminhos possíveis.
## Comandos básicos são:
  -IF
  if (condicao) {
    // código a ser executado se a condição for verdadeira
}
******************************************************************************************************************
  -IF ELSE
  if (condicao) {
    // código verdadeiro
} else {
    // código falso
}
******************************************************************************************************************
  -IF - ELSE IF - ELSE
  if (condicao1) {
    // bloco 1
} else if (condicao2) {
    // bloco 2
} else {
    // bloco final
}
******************************************************************************************************************
  -SWITCH CASE
  switch (variavel) {
    case valor1:
        // código
        break;

    case valor2:
        // código
        break;

    default:
        // código se nenhum caso for atendido
}
